## Anatomy of an Express app
### build checklist
- npm init
- install dependencies
  - express
  - mongoose
  - dotenv
  - ejs
- boot express
- boot mongoose
  - set .env
  - create and set connection
  - /models -> routes
- install middlewares
  - urlencoded
  - static
  - catch 404s
- set home page
- define routes
  - GET "/"
  - GET "/definitions"
  - GET "/new"
  - POST "/definitions"
- set and list server:port
